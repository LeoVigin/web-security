A small blog using CRUD through admin's rights.

## Installation & Start
- Download the folder or clone the repository using
``` php
git clone https://github.com/LeoVigin/web-security.git
```
- Open the project in any local development server.
- Install the database by copying and pasting the script from ```db.sql``` into your local database.
- Access the project through your localhost.

## Paths
Main page: ```index.php``` <br>
Sign Up: ```signup.php``` <br>
Log In: ```login.php``` <br>
Admin: ```admin.php``` (accessible only to administrators)
